# designbyark.com

### Technology Stack

- ReactJS
- Bootstrap 4
- HTML

### Installation

- Clone the repo

```shell
git clone https://github.com/mrarslanark/mrarslanark.github.io
```

- Run `npm install` to install dependencies
- Run `npm start` to run the project
